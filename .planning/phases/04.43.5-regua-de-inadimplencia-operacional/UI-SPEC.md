# UI Spec

## Tasks / Work Queue

Itens financeiros aparecem no modo execucao com:

- aluno;
- turno preferido;
- dominio Financeiro;
- severidade;
- valor e dias de atraso no motivo;
- mensagem pronta;
- CTA operacional.

## Acoes Rapidas

Para finance:

- Abrir WhatsApp;
- Copiar mensagem;
- Prometeu pagar;
- Pagamento confirmado;
- Link enviado;
- Sem resposta;
- Contestou cobranca;
- Encaminhar gerente;
- Adiar.

## Dashboard Financeiro

Adicionar bloco "Regua de inadimplencia":

- valor vencido;
- alunos inadimplentes;
- tasks abertas;
- recuperado 30 dias;
- breakdown D+1, D+3, D+7, D+15, D+30;
- botao "Atualizar regua".

## Estados

- Loading com skeleton.
- Erro explicito sem confundir com "sem dados".
- Empty state quando nao houver atraso.
