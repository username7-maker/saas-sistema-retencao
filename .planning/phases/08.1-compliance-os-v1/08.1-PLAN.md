# Phase 8.1 Plan - Compliance OS V1

## Objetivo

Entregar a fundacao operacional de compliance/LGPD para o piloto: consentimentos, termos e contratos registrados como historico auditavel, visiveis no perfil do aluno e consumidos pelo contexto canonico lead-to-member.

## Escopo

- Criar entidade append-only para consentimentos do membro.
- Registrar aceite, revogacao e vencimento operacional.
- Expor endpoints em `/lgpd/member/{member_id}/consents`.
- Integrar exportacao LGPD com historico de consentimentos.
- Fazer o contexto canonico usar a nova fonte antes de cair no fallback de `member.extra_data`.
- Adicionar painel simples no drawer de membro para recepcao/gestao registrar aceite/revogacao.

## Fora do escopo

- Assinatura digital com provider externo.
- Armazenamento de arquivo/documento assinado.
- Templates juridicos editaveis.
- Campanhas automaticas baseadas em consentimento.

## Validacao

- Testes unitarios de summary/append-only.
- Regressao do contexto lead-to-member.
- Build frontend.
- Spec Kit check.
