# 08.8 - AI First Review Center - Context

## Problema
As fases AI First recentes criaram varias fontes de rascunho supervisionado:

- `8.4` agente IA de atendimento na Kommo.
- `8.5` Personal IA para orientacao tecnica.
- `8.6` video de movimento supervisionado.
- `8.7` Personal IA do aluno via Kommo.

Cada fonte tem API e UI propria. Isso funciona para validar slices, mas fica ruim para operacao real: a equipe precisa saber o que revisar agora sem abrir varias telas.

## Decisao
Criar uma Central de Revisao AI First que agrega rascunhos, bloqueios, escalamentos e reviews tecnicos em um unico lugar.

## Nao objetivos
- Nao criar autoenvio.
- Nao substituir Work Queue.
- Nao criar tabela nova na V1.
- Nao apagar as filas especificas existentes.

## Fontes V1
- `AutopilotAction.action_type=kommo_draft_reply`
- `AutopilotAction.action_type=personal_ai_guidance_draft`
- `AutopilotAction.action_type=student_personal_ai_kommo_draft`
- `AutopilotAction.action_type=movement_video_feedback_draft`
- `MovementVideoReview` pendente, bloqueado ou aprovado para envio supervisionado.

## Principio operacional
AI First so vira produtivo se existir uma mesa de controle: revisar, aprovar, preparar na Kommo, rejeitar ou escalar.
