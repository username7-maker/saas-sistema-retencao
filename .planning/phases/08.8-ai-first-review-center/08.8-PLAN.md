# 08.8 - AI First Review Center - Plan

## Objetivo
Criar uma central unica para revisar rascunhos e reviews AI First antes de qualquer envio ao aluno ou lead.

## Entregas
1. Spec Kit `025-ai-first-review-center`.
2. Obsidian atualizado com decisao de produto.
3. API `/api/v1/ai/review-center/items` agregando rascunhos e reviews.
4. API `/api/v1/ai/review-center/metrics` com contadores por origem/status.
5. Acoes supervisionadas: `prepare-kommo` e `reject`.
6. UI `/ai/review-center` com filtros, metricas, lista e inspector.
7. Navegacao para owner/manager/receptionist/trainer/salesperson conforme escopo.

## Escopo backend
- Criar schemas `ai_review_center.py`.
- Criar service `ai_review_center_service.py`.
- Criar router `ai_review_center.py`.
- Reutilizar services existentes de AI Service Agent, Personal IA, Student Personal IA e Movement Video.
- Respeitar tenant e permissao por papel.

## Escopo frontend
- Criar `aiReviewCenterService`.
- Criar `AiReviewCenterPage`.
- Adicionar rota e item de menu.
- Mostrar origem, aluno/lead, status, resumo, rascunho, bloqueios e CTA.

## Guardrails
- V1 continua `draft-only`.
- Item bloqueado ou sensivel nao pode ser preparado para envio.
- Qualquer preparo Kommo continua sendo handoff/tarefa, nao autoenvio.
- Rejeicao registra trilha auditavel.

## Validacao
- Testes focados de service/API.
- Build frontend.
- Validacao manual: criar/listar rascunhos, preparar Kommo, rejeitar e conferir filtros.
