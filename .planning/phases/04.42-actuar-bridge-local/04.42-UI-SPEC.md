# Phase 4.42 UI Spec - Actuar Bridge local

## Settings > Actuar

- mostrar o modo `Ponte local`
- explicar que a automacao roda na estacao da academia e usa a sessao local do navegador
- exibir:
  - numero de estacoes pareadas
  - ultima atividade
  - status online/offline
  - botao para gerar codigo de pareamento
  - botao para revogar estacao

## UX tone

- honesta
- operacional
- sem prometer que a aba do navegador sera automatizada ate a estacao estar online

## Empty states

- sem estacao pareada: mostrar CTA de pareamento
- estacao offline: mostrar aviso claro para o owner
- modo `csv_export`: manter o fallback manual assistido explicito
