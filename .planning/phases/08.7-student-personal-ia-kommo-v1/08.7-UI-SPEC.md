# 08.7 - Student Personal IA Kommo V1 - UI Spec

## Settings
Adicionar aba `Aluno IA` para owner/manager:
- status: Kommo, Draft-only, Autoenvio desligado.
- toggles: habilitar, exigir Kommo, texto tecnico, video, consentimento de comunicacao, consentimento de imagem.
- limites: rascunhos por dia e cooldown apos atendimento humano.
- lista de entradas recentes.

## Work Queue
Itens `student_personal_ai` aparecem na fila tecnica:
- badge `Aluno Kommo`.
- badge `Personal IA`.
- badge `Video` quando aplicavel.
- CTA `Preparar resposta Kommo`.
- bloqueios explicitos quando nao pode responder.

## Estados
- `draft_ready`: pronto para preparar na Kommo.
- `blocked`: guardrail impediu resposta.
- `escalated`: humano deve assumir.
- `awaiting_outcome`: preparado na Kommo, aguardando retorno.

## Nao Fazer
- Nao mostrar como envio automatico.
- Nao prometer analise automatica do video.
- Nao esconder bloqueios de consentimento ou Kommo nao pronta.
