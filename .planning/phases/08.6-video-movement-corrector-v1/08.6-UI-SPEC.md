# Phase 8.6 UI Spec - Video Movement Corrector V1

## Primary surfaces

- Coach Workspace.
- Perfil 360 do aluno.
- Tasks / fila tecnica do professor.
- Settings.

## Coach Workspace

Adicionar card `Analise de movimento`.

Conteudo:

- aluno;
- exercicio;
- status do video;
- qualidade do video;
- resumo assistivo;
- pontos observados;
- bloqueios;
- feedback sugerido;
- historico de revisao.

CTA principal:

- `Revisar video`.

CTAs secundarios:

- `Gerar observacao assistiva`;
- `Aprovar feedback`;
- `Editar feedback`;
- `Preparar na Kommo`;
- `Criar task tecnica`;
- `Rejeitar`.

## Perfil do aluno

Adicionar secao `Movimento e tecnica`.

Mostrar:

- ultimos reviews;
- exercicios revisados;
- feedbacks aprovados;
- bloqueios/safety;
- relacao com tasks tecnicas.

## Empty state

Texto:

`Nenhum video tecnico analisado ainda. Use esta area para revisar execucao com supervisao do professor.`

## Safety copy

Sempre exibir:

`Analise assistiva. Nao substitui professor, medico ou fisioterapeuta. O professor revisa antes do aluno receber.`

## Visual constraints

- Nao usar linguagem de diagnostico.
- Nao destacar "erro" como julgamento definitivo; usar "ponto de atencao".
- Dark mode obrigatorio.
- Mobile deve permitir leitura, mas V1 e desktop-first.
