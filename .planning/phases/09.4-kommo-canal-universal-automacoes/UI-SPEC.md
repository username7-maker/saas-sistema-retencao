# UI Spec

## Settings > Kommo
- Mostrar checklist "Cobertura das automacoes".
- Cada dominio deve exibir status: pronto, desativado ou campo faltante.
- Quando canal principal for Kommo, avisar que automacoes usam Salesbot e WhatsApp e fallback.

## Operacao
- Botao "Enviar pelo canal principal" continua existindo.
- Quando falhar rota Kommo, mensagem de erro deve explicar o campo faltante.
- "Preparar na Kommo" permanece como fallback legado.
