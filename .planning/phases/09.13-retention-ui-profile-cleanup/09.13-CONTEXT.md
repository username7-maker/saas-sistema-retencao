# 09.13 Context - Retention UI Profile Cleanup

## Problem
As telas de retencao carregam informacao operacional util, mas estao densas e exigem rolagem ate as acoes. A aba Retencao do drawer de aluno tem superficies claras herdadas que ficam ilegíveis no tema escuro. No Perfil 360, Cordex Coach e Video de Movimento devem sair temporariamente da superficie principal para reduzir ruido.

## Scope
- Drawer/playbook de retencao.
- Aba Retencao no drawer de aluno.
- Perfil 360, apenas removendo os paineis Coach/Motion do render.

## Constraints
- Sem mudanca de API, banco, RBAC ou rotas.
- Preservar funcionalidades existentes.
- Usar design system Cordex/lovable atual.
- Registrar Spec Kit, GSD e Obsidian.
