# 09.2 - Cordex Design System V1

## Contexto

O rebrand 09.1 trocou a marca visivel para Cordex Gym OS, mas o produto ainda carregava a sensacao visual de template SaaS roxo. Esta fase transforma a troca de nome/logo em uma identidade real: dark tecnico, nucleo azul Cordex, orbitas discretas e componentes base mais coerentes.

## Decisao

Cordex Gym OS usa identidade dark tecnica, nucleo azul e orbitas como linguagem visual. Classes tecnicas `lovable-*` permanecem por compatibilidade; mudam os valores dos tokens e as superficies visiveis.

## Escopo

- Tokens visuais em `lovable-theme.css`.
- Constantes semanticas em `brand.ts`.
- Assets Cordex limpos para logo, mark, watermark e favicon.
- Login, reset, shell/sidebar, navegacao e componentes base.
- Labels visiveis de IA para linguagem Cordex.
- Relatorios premium/bioimpedancia com acento azul Cordex.

## Fora de Escopo

- Renomear rotas, endpoints, tabelas, migrations, cookies, storage keys ou schemas.
- Redesign completo de fluxos.
- Mudanca de permissao, tenant scoping ou banco.
- Troca de ProGym por Cordex em contexto de academia/tenant.
