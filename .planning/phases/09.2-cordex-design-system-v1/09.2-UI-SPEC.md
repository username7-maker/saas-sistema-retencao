# UI Spec 09.2 - Cordex Design System V1

## Linguagem Visual

- Fundo: carvao quase preto com profundidade azulada.
- Superficies: preto azulado translucido, borda grafite/azul de baixo contraste.
- Acento principal: azul nucleo Cordex.
- Acento secundario: off-white.
- Motivo visual: orbita/atomo como watermark discreto, nunca como ruido.

## Componentes

- Botoes primarios usam gradiente azul nucleo com texto escuro.
- Cards usam gradiente sutil, borda fina e glow azul moderado.
- Inputs usam superficie tecnica escura e foco azul.
- Badges devem ser mais finos, menos saturados e mais legiveis.
- Tabs ativas usam energia azul; roxo deixa de dominar.

## Telas Prioritarias

- Login e reset: marca Cordex, mark limpo, watermark orbital.
- Shell/sidebar: mark Cordex e tagline curta.
- Tasks/Work Queue: componentes herdam tokens e labels Cordex.
- Configuracoes: tabs de IA renomeadas.
- Revisao Cordex: texto e empty states com linguagem Cordex.
- Bioimpedancia: PDF/tela usam acento azul e marca Cordex.

## Restrições

- Nao alterar layout estrutural de fluxos.
- Nao trocar ProGym quando representar academia.
- Nao usar a imagem quadrada original inteira em icones pequenos.
