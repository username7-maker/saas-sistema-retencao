# 04.42.3 Plan

## Objetivo

Substituir o PDF atual de bioimpedancia por laudos premium de avaliacao, com dois formatos complementares:
`Resumo do aluno` e `Relatorio tecnico`.

## Escopo

1. Criar payload premium de avaliacao com:
   - dados do aluno
   - composicao corporal
   - perimetria quando existir
   - avaliacao anterior para comparativo
   - leitura resumida
   - leitura tecnica
   - plano de acao
2. Implementar dois templates premium:
   - resumo do aluno
   - relatorio tecnico
3. Adicionar endpoints dedicados:
   - `GET /members/{member_id}/body-composition/{evaluation_id}/pdf`
   - `GET /members/{member_id}/body-composition/{evaluation_id}/technical-pdf`
4. Esconder secoes opcionais ausentes sem quebrar o layout.
5. Reaproveitar canais de entrega existentes:
   - download
   - WhatsApp
   - handoff Kommo quando aplicavel

## Fora de escopo

- criar perimetria nova no dominio se o dado ainda nao existir
- abrir novo canal de entrega

## Requisitos

- `RPTA-01`: resumo premium para aluno
- `RPTA-02`: relatorio tecnico premium
- `RPTA-03`: fallback de secoes opcionais
- `RPTA-04`: entrega pelos canais existentes

## Gates de saida

- um relatorio premium de avaliacao real e validado no piloto
- o layout funciona com e sem perimetria
- o layout funciona com e sem exame anterior
- a leitura do aluno e a leitura tecnica ficam claramente distintas
