# Phase 3 UI Spec

## Automations

- Botao `Regras Padrao` visivel apenas para `owner`.
- Acao de excluir regra visivel apenas para `owner`.
- Se uma acao owner-only escapar e receber `403`, o toast precisa explicar isso explicitamente.

## Users

- Botao principal de desligamento: `Desativar`.
- Dialogo explica perda de acesso agora e possibilidade de reativacao depois.
- `Reativar` continua disponivel quando o usuario estiver inativo.
