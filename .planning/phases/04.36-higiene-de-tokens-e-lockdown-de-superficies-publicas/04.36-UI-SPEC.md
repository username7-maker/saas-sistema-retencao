# 04.36 UI Spec

## Superficies afetadas

- fluxo de reset de senha
- telas que abrem websocket
- erros de readiness/operacao expostos via UI
- eventuais superficies publicas com mensagens de indisponibilidade

## Regras de UX

- nenhum link de usuario final deve carregar token sensivel em query string
- a UI precisa continuar simples no reset de senha mesmo com token fora da query
- estados publicos devem falhar de forma controlada, sem vazar detalhe tecnico
- mensagens de endpoint publico bloqueado precisam ser honestas e curtas
