# 08.10 - UI Spec

## Principio
A UI deve deixar claro quando um texto veio de agente especialista, sem passar a impressao de decisao autonoma.

## Badges
- `Agente especialista`
- `Prompt vX`
- `Modelo: gpt-5.4-mini`
- Safety profile quando estiver em contexto de revisao.

## Bioimpedancia / Avaliacao
- Manter separacao entre `Resumo para professor`, `Resumo para aluno`, `Direcao sugerida` e `Evidencias`.
- Mostrar metadata em bloco discreto, abaixo do painel de IA.

## Central de Revisao IA
- Mostrar prompt/modelo no inspector.
- O revisor deve continuar aprovando, editando, escalando ou rejeitando antes de qualquer envio.

## Estados
- Fallback deterministico deve ser visivel quando metadata indicar `fallback_used=true`.
- Nenhuma tela deve prometer autoenvio nesta fase.
