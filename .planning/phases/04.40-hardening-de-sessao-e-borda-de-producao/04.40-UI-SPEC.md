# 04.40 UI Spec

## Superficies afetadas

- login e refresh de sessao
- carregamento inicial da SPA
- borda publica do frontend e da API

## Regras de UX

- o endurecimento de sessao nao pode piorar drasticamente a experiencia de login
- expiracao/refresh precisam continuar previsiveis para o usuario
- erros de configuracao de build/producao devem aparecer cedo para o time, nao para o usuario final
