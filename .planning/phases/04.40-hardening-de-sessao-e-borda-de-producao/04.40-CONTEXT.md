# 04.40 Context

## Problema

Mesmo com o piloto funcionando, a borda de producao ainda esta abaixo do ideal:

- tokens em `localStorage`
- headers de seguranca ausentes
- CSP inexistente
- defaults de build que ainda podem apontar para localhost

Isso nao e o bug mais visivel do dia a dia, mas e o tipo de risco que vira incidente serio.

## Objetivo

Diminuir o blast radius de XSS/sessao e estabelecer um baseline de borda segura para producao.

## Fora do escopo

- migracao para um gateway de API dedicado
- revisao completa de auth federado
- reestruturacao total do frontend
