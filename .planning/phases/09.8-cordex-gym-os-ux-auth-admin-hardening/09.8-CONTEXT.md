# 09.8 - Contexto

## Objetivo
Corrigir friccoes reais do Cordex Gym OS publicado: poluicao visual, reset de senha inacessivel/quebrado e criacao de usuarios com atrito alto.

## Escopo desta entrega
- Limpeza visual conservadora no app inteiro, sem remover rotas.
- Login com recuperacao de senha.
- Reset de senha com feedback operacional honesto.
- Criacao de usuario por senha provisoria auto-gerada.
- Reset administrativo de senha com RBAC.

## Decisoes travadas
- Projeto alvo: `saas-sistema-retencao-github-latest` e alias `https://saas-frontend-pearl.vercel.app`.
- Politica visual: conservadora, mantendo modulos e RBAC.
- Criacao de usuario: senha provisoria auto-gerada.
- Reset: login + e-mail + admin.
- Sem deploy automatico nesta fase.
