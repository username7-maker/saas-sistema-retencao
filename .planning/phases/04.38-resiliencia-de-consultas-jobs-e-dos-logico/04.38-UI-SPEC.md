# 04.38 UI Spec

## Superficies afetadas

- Tasks
- Imports
- dashboards com listagens de apoio
- operacoes que dependem de jobs assincronos

## Regras de UX

- a UX pode ganhar estados de processamento mais honestos em vez de travar tentando fazer tudo inline
- o usuario nao deve perder clareza por causa da otimizacao tecnica
- limites operacionais precisam virar mensagens claras, nao timeouts silenciosos
