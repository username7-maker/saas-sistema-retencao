# 04.39 Context

## Problema

O isolamento multi-tenant esta bom, mas depende de:

- `with_loader_criteria`
- `include_all_tenants`
- `set_unscoped_access`
- disciplina de router/service para commit

Isso funciona hoje, mas e fragil para manutencao futura.

## Objetivo

Reduzir os bypasses manuais e padronizar a fronteira transacional dos fluxos criticos.

## Fora do escopo

- trocar completamente a estrategia de multi-tenant
- introduzir RLS nativa nesta rodada
- reescrever todos os services do projeto
