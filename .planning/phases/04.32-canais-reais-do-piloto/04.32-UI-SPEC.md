# 04.32 UI Spec

## Objetivo da interface

As superficies de canais reais do piloto precisam comunicar estado operacional verdadeiro, nao "aparencia de integracao".

## Settings > WhatsApp

- mostrar estado atual com copy curta:
  - `Conectado`
  - `Aguardando QR`
  - `Falha de configuracao`
- mostrar qual instancia esta ativa
- mostrar se o ultimo envio auditavel foi bem sucedido

## Settings > Actuar

- mostrar estado efetivo do sync:
  - `CSV export/manual`
  - `Assisted RPA configuravel`
  - `Automatico validado`
- destacar ausencia de `base_url`, `usuario` ou `senha` como bloqueio real
- o botao de teste deve mostrar resultado honesto:
  - conexao validada
  - automatico indisponivel por falta de credenciais
  - falha de autenticacao/acesso

## Guardrails de copy

- nao usar `integrado` quando o estado for apenas `csv_export`
- nao usar `automatico pronto` sem `test-connection` verde
- nao ocultar bloqueio externo de credenciais reais
