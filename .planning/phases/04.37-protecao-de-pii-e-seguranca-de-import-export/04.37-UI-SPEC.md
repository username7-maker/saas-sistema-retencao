# 04.37 UI Spec

## Superficies afetadas

- imports de membros e catraca
- exports CSV
- mensagens de erro de importacao
- telas que exibem ou operam sobre dados sensiveis

## Regras de UX

- operador deve receber erro claro quando o arquivo exceder limites de seguranca
- exports continuam abrindo normalmente para o usuario, sem alterar visualmente o valor exibido no sistema
- a reducao de PII em logs/auditoria nao pode quebrar troubleshooting basico
