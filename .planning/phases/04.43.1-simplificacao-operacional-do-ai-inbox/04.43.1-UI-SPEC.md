# 04.43.1 UI-SPEC

## Direcao de interface

- visao padrao: operador-first
- fila unica
- texto curto
- um CTA principal por item
- detalhes analiticos recolhidos

## Lista

Mostrar por item:

- aluno/lead
- severidade
- dominio
- turno preferido
- acao principal curta
- motivo operacional curto

Esconder por padrao:

- score detalhado
- owner
- canal
- approval state
- execution state
- explainability longa

## Inspector

Ordem visual:

1. Fazer agora
2. Mensagem pronta ou resumo da acao
3. CTA principal
4. Registrar resultado
5. Detalhes analiticos recolhidos
