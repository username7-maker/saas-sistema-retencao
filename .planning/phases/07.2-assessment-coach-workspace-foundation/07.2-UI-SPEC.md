# Phase 7.2 UI Spec - Coach Workspace V1

## Principio

A tela deve responder rapidamente:

1. quem o professor precisa acompanhar;
2. por que esse aluno apareceu;
3. o que fazer agora;
4. qual resultado registrar.

## Entrada recomendada

V1 pode aparecer em um destes caminhos, priorizando menor ruptura:

- `Tarefas > Professor`, evoluindo a fila tecnica existente; ou
- `Avaliacoes > Coach Workspace`, como cockpit tecnico.

Se ambos existirem, `Tarefas > Professor` continua sendo a entrada diaria de execucao, e `Avaliacoes > Coach Workspace` vira apoio tecnico.

## Layout

### Header

- Titulo: `Fila tecnica do professor`.
- Subtitulo: `Treinos, avaliacoes, bioimpedancia e feedbacks do seu turno`.
- Filtros rapidos:
  - `Meu turno`
  - `Todos os turnos` para manager/owner
  - `Treino`
  - `Feedback`
  - `Reavaliacao`
  - `Bioimpedancia`
  - `Ajuste tecnico`

### Card de item

Mostrar apenas:

- aluno;
- turno preferido;
- lane tecnica;
- acao principal;
- motivo curto;
- prazo;
- ultima avaliacao/bioimpedancia quando relevante.

Nao mostrar por padrao:

- score detalhado;
- payload tecnico bruto;
- explicacoes longas;
- dados financeiros/comerciais;
- retencao comum.

### Inspector

Blocos:

1. `Fazer agora`
2. `Evidencias tecnicas`
3. `CTA principal`
4. `Registrar resultado`
5. `Abrir perfil 360`
6. `Ver detalhes` recolhido

### CTAs principais

- `Verificar treino`
- `Registrar feedback`
- `Agendar reavaliacao`
- `Revisar bioimpedancia`
- `Ajustar treino`
- `Abrir Perfil 360`

### Outcomes rapidos

- `Treino entregue`
- `Treino ausente`
- `Treino ajustado`
- `Feedback positivo`
- `Precisa ajuste`
- `Reavaliacao agendada`
- `Encaminhar recepcao`
- `Sem acao tecnica agora`

## Empty states

- Se professor nao tem turno: `Login sem turno operacional. Configure o turno do usuario para montar a fila tecnica.`
- Se nao ha itens: `Nenhuma acao tecnica no seu turno agora.`
- Se falta dados: `Sem base tecnica suficiente para montar recomendacao.`

## Responsividade

- Desktop: lista + inspector lado a lado.
- Mobile/tablet: lista primeiro, detalhe em drawer ou rota dedicada.

## Regras visuais

- Badge `Professor`.
- Badge de lane tecnica.
- Badge de turno.
- Badge `Sem base` quando recomendacao depende de dado ausente.
- Nao usar linguagem de IA autonoma.
