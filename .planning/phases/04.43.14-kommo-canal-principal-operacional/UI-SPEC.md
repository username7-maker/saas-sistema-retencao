# UI Spec

## Settings

- Aba Kommo mostra canal principal: Kommo, WhatsApp ou Manual.
- Exibe fallback WhatsApp ativo.
- Mostra flags: envio confirmado por operador e auto-close por evento.

## Work Queue

- Card mostra badge `Kommo` quando o canal resolvido for Kommo.
- CTA principal muda para `Enviar para Kommo`.
- Estado aguardando resultado mostra `Aguardando resposta na Kommo`.
- Se fallback ocorrer, mostrar badge `Fallback WhatsApp`.

## AI Inbox e Tasks

- Textos de ação devem usar canal resolvido, não presumir WhatsApp.
