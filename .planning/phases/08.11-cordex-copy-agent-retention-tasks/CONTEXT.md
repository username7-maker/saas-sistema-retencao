# 08.11 - Cordex Copy Agent Retencao e Tasks

## Contexto
Mensagens de retencao, onboarding e tasks operacionais ainda eram majoritariamente templates fixos. O sistema ja possuia Prompt Registry para agentes tecnicos, mas a operacao diaria continuava sem copy contextual por IA.

## Decisao
Mensagens operacionais passam a ter rascunho especialista quando seguro. Templates atuais continuam fallback. Nenhum fluxo ganha autoenvio nesta fase.

## Escopo
- Retencao e reativacao.
- Tasks humanas com aluno ou lead.
- Onboarding D0-D30 na Central Cordex.
- Financeiro/comercial como prompts disponiveis, com bloqueio de sensiveis.

## Fora de escopo
- Envio automatico.
- Mudanca de banco obrigatoria.
- Remocao de templates existentes.
