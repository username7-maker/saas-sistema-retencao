# saas-frontend

Frontend React 18 + TypeScript do AI GYM OS.

## Rodar local

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Variaveis

- `VITE_API_BASE_URL=http://127.0.0.1:8000`

## Rotas principais

- `/dashboard/executive`
- `/dashboard/operational`
- `/dashboard/commercial`
- `/dashboard/financial`
- `/dashboard/retention`
- `/crm`
- `/tasks`
- `/notifications`

## E2E

```bash
npm run test:e2e
```
