# Spec 033 - Cordex Copy Agent Retencao e Tasks

## User Story
Como operador da academia, quero receber mensagens prontas melhores e contextualizadas para abordar alunos e leads sem precisar escrever tudo do zero.

## Acceptance Criteria
- Rascunhos de retencao e tasks usam agente especialista quando seguro.
- Templates atuais sao fallback seguro.
- Casos sensiveis nao geram copy por IA.
- UI mostra origem da mensagem e permite regenerar.
- Nenhum envio automatico e ativado.
