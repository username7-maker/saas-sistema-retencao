# Implementation Plan: Cordex Design System V1

## Sequencia

1. Criar/organizar assets Cordex limpos.
2. Atualizar `brand.ts` com tokens semanticos.
3. Atualizar `lovable-theme.css` e `tailwind.config.js`.
4. Ajustar componentes base.
5. Ajustar login, reset e layout.
6. Atualizar labels visiveis de modulos IA.
7. Ajustar relatorios premium/bioimpedancia.
8. Validar lint/build e busca final.

## Compatibilidade

Manter nomes `lovable-*` e `ai_gym_*` quando tecnicos. Esta fase nao altera contratos persistentes.
